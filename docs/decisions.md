Foundations gets one module per domain (5) plus a capstone (6); Professional's 7 domains cluster into 3 real thematic groups (design / build & validate / govern & deliver) plus a capstone (4). See `docs/workshop-design.md`'s module arc tables. |
| 2026-07-14 | Two-tier module gate: hands-on Claude Code artifact, then closed-book no-AI practice checkpoint | Named design tension: the workshop's method (agent-native) is the opposite of the real exam's condition (proctored, closed-book, no AI). Proposed resolution is not either/or but both, in sequence, per module — see `docs/design-tension.md`. **Stated hypothesis, not yet evidenced** — no module content exists yet and neither real exam has been sat. Practice questions are always written originally against the published blueprint, never copied from real/leaked exam content or Anthropic's own official sample questions. |
| 2026-07-14 | Dogfooding commitment: coderturtle will personally sit the real CCA-F exam after Modules 01-06 exist, and CCAR-P after Modules 07-10 exist | Same discipline as Borrow Native's Ardan Labs commitment — stated intent, not yet evidenced. |
