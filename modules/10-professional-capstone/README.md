# Module 10: Professional Capstone — Sit-Ready for CCAR-P

## The question this module answers

Are you actually ready to sit the real Claude Certified Architect – Professional exam?

## Where it sits in the arc

Tenth and final module, closing Part 2 (Architect Professional) and the workshop's whole arc. Hard prerequisite: all of [Module 07](../07-solution-design-context-strategy/README.md)–[Module 09](../09-governance-stakeholders/README.md) — this module synthesizes all seven CCAR-P domains rather than testing any one in isolation. See [`modules/README.md`](../README.md).

## Learning objectives (placeholder — finalized when content is authored)

- Diagnose, under real exam-scenario framing, which of the seven CCAR-P domains a given end-to-end business problem actually spans.
- Complete a full closed-book mock exam at real exam pacing (120 minutes, 63 items) and correctly self-score against the 720/1000 threshold.
- Defend a complete architecture proposal in writing against a real stakeholder objection — the kind a client or exec would actually raise, not a softball.

## Exercise material to draw from (not a spec — Coachgremlin authors the real exercise later)

The CCAR-P Exam Guide itself, in full: its stated blueprint across all 7 domains and its 3 published sample questions, used as a style/difficulty reference only — never copied or lightly reworded into this module's own mock exam (see [`docs/design-tension.md`](../../docs/design-tension.md)'s Constraint section). See [`docs/workshop-design.md`](../../docs/workshop-design.md).

## Required gate (placeholder — shape decided now, real rubric written later)

- **Deterministic tier (hands-on, with Claude Code):** a real end-to-end architecture review — spanning design, integration/evaluation, and governance/delivery — defended in writing against a real seeded stakeholder objection (e.g. a cost, compliance, or reliability challenge that would actually change the design).
- **Exam-condition tier (closed-book, without Claude Code):** a full mock exam mirroring the real exam's own structure — 63 items, multiple-choice and multiple-response, 120 minutes, 720/1000 to pass. Like Module 06, this is the one module where the checkpoint format matches the real exam exactly rather than the workshop's smaller per-module default.

## Takeaway

A personal CCAR-P exam-day prep sheet, same discipline as Module 06's — built from this specific attempt's weak spots, not a generic study guide. Packaged by Coachgremlin once the rubric is met.

## Stop condition (placeholder)

The learner's stakeholder-objection defense is real and specific, the mock exam is completed at real pacing and scored, and Coachgremlin confirms both before this module — and the workshop — counts as done. Passing the mock exam is not the same claim as passing the real one — see [`docs/workshop-design.md`](../../docs/workshop-design.md)'s dogfooding commitment for how this workshop tests that claim for real.
